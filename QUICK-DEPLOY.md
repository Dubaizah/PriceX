# Quick Deploy Guide for PriceX

## Option 1: One-Click Deploy (Easiest!)

1. **Copy this link:**
```
https://vercel.com/new/import?repository=github.com/Dubaizah/PriceX&env=NEXT_PUBLIC_APP_NAME=PriceX
```

2. **Paste it in your browser** (while logged into Vercel)

3. **Click "Deploy"** - that's it!

---

## Option 2: Manual (If Option 1 doesn't work)

1. Go to: https://vercel.com/dashboard/new

2. Under "Import Project", click **"Import Project"**

3. It will ask for a repo - type: **Dubaizah/PriceX**

4. Click **"Import"**

5. Scroll down and click **"Deploy"**

---

## What's Happening?

✅ Deploying your code from GitHub
✅ Building the Next.js app
✅ Setting up free SSL certificate
✅ Giving you a free URL (pricex.vercel.app)
